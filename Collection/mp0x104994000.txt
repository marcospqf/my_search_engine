                <h1>
                    Como habilitar o JavaScript no seu navegador</h1>
                <h2 class="intro">
                    Atualmente quase todas as páginas da internet contém JavaScript, uma linguagem de
programação que é executada no navegador do visitante. Ela torna as páginas funcionais
para propósitos específicos, e se estiver desabilitada por alguma razão, o conteúdo ou
funcionalidade da página pode se tornar limitado ou indisponível. Aqui você encontra 
instruções sobre como habilitar (ativar) o JavaScript nos cinco navegadores mais comumente
usados.</h2>

                    <a id="ieicon" class="ie48" href="#ie">Internet Explorer</a>

                    <h3 id="ie">
                        <span class="icon">&nbsp;</span>Internet Explorer
                    </h3>
                    <ol>
                        <li>
                            No navegador clique no &#237;cone &quot;Ferramentas&quot; e selecione &quot;Op&#231;&#245;es da Internet&quot;</li>
                        <li>
                            Na janela &quot;Internet Options&quot; selecione a guia &quot;Security&quot;.</li>
                        <li>
                            Na guia &quot;Security&quot; clique no bot&#227;o &quot;Custom level...&quot;.</li>
                        <li>
                            Quando a janela de di&#225;logo &quot;Security Settings - Internet Zone&quot; abrir, procure pela se&#231;&#227;o &quot;Scripting&quot;.</li>
                        <li>
                            No item &quot;Active Scripting&quot; selecione &quot;Enable&quot;.</li>
                        <li>
                            Quando a janela &quot;Warning!&quot; aparecer perguntando &quot;Are you sure you want to change the settings for this zone?&quot; selecione &quot;Yes&quot;.</li>
                        <li>
                            Na janela &quot;Internet Options&quot; clique no bot&#227;o &quot;OK&quot; para fech&#225;-la.</li>
                        <li>
                            Clique no bot&#227;o &quot;Refresh&quot; do navegador para recarregar a p&#225;gina.</li>
                    </ol>

                            Clique no bot&#227;o de &quot;OK&quot; para fech&#225;-lo.</li>

                            Feche a aba &quot;Settings&quot;.</li>
                        <li>
                            Clique no bot&#227;o &quot;Reload this page&quot; no navegador para atualizar a p&#225;gina.</li>
